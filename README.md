## Installation

### Mac

1. Clone the repo

```bash
	git clone git@gitlab.com:seakaart/chameleon-2.git
```
2. Install `q` and `pyqt`
	
```bash 
	brew install q pyqt
```

3. Change into the directory

```bash 
	cd chameleon-2/
```

4. Copy and paste the following into the terminal

 ```bash
 	pyuic5 design.ui -o design.py
 ```
